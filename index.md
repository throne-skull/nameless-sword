# Intro
Nameless Sword is a free SRD for playing classic/OSR adventures, essentially a small-scale wargame with exploration procedures and a few rules to handle character advancement. It is the latest iteration of [Ensifera](https://dave-craigen.itch.io/ensifera). 

### Main Inspirations
* [Chainmail](https://en.wikipedia.org/wiki/Chainmail_(game))
* [Delving Deeper](https://ddo.immersiveink.com/dd.html#)
* [Wolves upon the Coast](https://lukegearing.blot.im/wolves-upon-the-coast)
* [Delta](https://deltasdnd.blogspot.com)
* [Press the beast](https://pressthebeast.bearblog.dev/blog/)
* [Meatheads](https://halforc.itch.io/meatheads)
* [Cavegirl](https://cavegirlgames.blogspot.com)
